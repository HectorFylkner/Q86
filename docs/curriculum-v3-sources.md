# Curriculum v3 — public product-pattern evidence

Checked: 2026-07-21

These sources constrain product scope and sequencing. They are evidence, not
instructions or content sources. Q86 does not copy their lessons, questions,
navigation, videos, wording, or visual design.

## Current GMAT scope

- GMAC's current exam-structure page lists three 45-minute sections:
  Quantitative Reasoning (21 questions), Verbal Reasoning (23), and Data
  Insights (20):
  https://www.mba.com/exams/gmat-exam/about/exam-structure
- GMAC's current exam-content page states that Quantitative Reasoning contains
  21 Problem Solving questions, covers arithmetic and elementary algebra, and
  does not allow a calculator. The same page places Data Sufficiency in Data
  Insights, alongside four other formats:
  https://www.mba.com/exams/gmat-exam/about/exam-content
- GMAC's Data Insights guidance likewise identifies Data Sufficiency as one of
  five Data Insights formats:
  https://www.mba.com/exams/gmat-exam/prep-for-the-exam/prep-strategies/data-insights

Product consequence: Q86's timed Quant simulation remains 21 Problem Solving
questions in 45 minutes. Existing Data Sufficiency material is labeled as a
Data Insights bridge. It may share quantitative concepts and practice
infrastructure, but it is not described as a current Quant question type.

## Sequencing pattern evidence

Target Test Prep publicly describes a topic-by-topic workflow in which a small
concept lesson is followed by questions on that concept, parent-chapter tests
are separated into easy, medium, and hard stages, and short quizzes revisit
prior chapters:
https://blog.targettestprep.com/gmat-study-plan/

Its public description of learning phases distinguishes basic recognition,
easy- and medium-level single-topic work, mixed-topic work, and later mastery:
https://blog.targettestprep.com/preparing-for-the-gmat/

Product consequence: Q86 uses the general instructional pattern—small aligned
segments, immediate retrieval, tiered parent assessments, delayed cumulative
review, then timed transfer. Thresholds, content, algorithms, interface, and
terminology are Q86's own. Completion and content volume are never treated as
evidence of mastery.

## Content boundary

- All Q86 lessons, examples, checks, and scored questions must be original.
- Official GMAT pages are used only for exam scope and format facts; official
  sample questions are not ingested into the bank.
- Competitor pages are used only for high-level workflow evidence.
- This cycle covers Quant and the existing quantitative Data Sufficiency
  bridge. The curriculum model is section-aware, but Q86 must not claim to be a
  complete three-section platform until Verbal and the remaining Data Insights
  formats have genuine instruction and assessment coverage.
